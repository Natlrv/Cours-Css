Sélecteurs

  • * (pour tout)
  • p (pour les balises p coder HTML "fonctionne avec toutes les balises")
  • .toto (pour les élémants de la classe : toto coder en HTML)
  • #titi (pour les éléments des ID : titi coder en HTML)

    • Les propriétés s'applique à tout les enfants et petits enfants et petits petits enfants...
    Exemple : si on code *{color:black;}
    tout sera écrit en noir
    mais si je code
    *{color:black;}
    Et p{color:blue;}
    Alors tout sera en noir sauf tout ce qui appartient a un paragraphe p
